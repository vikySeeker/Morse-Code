#Ineffecient Yet Simple Morse Code Generator using java

##Requirements:

Java JDK

##Instructions to Run:

javac MorseCode.java

java MorseCode -h

##Options:

###English to morse code:

    Example: java MorseCode 'hello world' 'welcome to morse code'

##Morse code to English

    Example: java MorseCode -r '.... . .-.. .-.. --- / .-- --- .-. .-.. -..' '.-- . .-.. -.-. --- -- . / - --- / .. ... .-.. .- -. -..'
